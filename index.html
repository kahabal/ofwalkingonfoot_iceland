📊 Google Sheets Sync

One-time setup:
1. sheets.google.com → New spreadsheet
2. Extensions → Apps Script → paste the script from the Security section below
3. Deploy → New Deployment → Web App · Execute as: Me · Access: Anyone
4. Copy URL → paste above

🔐 Security Token

Set a secret word here and in your Apps Script to block unauthorized writes.

Apps Script to use: var SECRET = 'your_token_here'; function doPost(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); var data = JSON.parse(e.postData.contents); if (data._token !== SECRET) return ContentService.createTextOutput('denied'); delete data._token; if (sheet.getLastRow() === 0) sheet.appendRow(Object.keys(data)); sheet.appendRow(Object.values(data)); return ContentService.createTextOutput('ok'); }
